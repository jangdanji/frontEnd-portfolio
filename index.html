자바스크립트 활용 쇼핑몰 프로젝트 (칼하트)

코딩 기간 : 2023년 5월 4일 ~ 6월 14일

HTML, CSS, JAVASCRIPT, JQUERY, SASS 사용

프로젝트 설명 및 후기

칼하트의 강점인 거칠고 힙한 브랜드 이미지를 살리면서 사용자의 이용성과 직관성을 높이기 위해 제작한 프로젝트입니다. 자바스크립트를 사용하여 동적인 웹 페이지를 구성하는 것에 초점을 맞추었습니다.

유저 페이지 : 회원가입, 로그인, 계정 찾기 등
메인 페이지 : 애니메이션, 슬라이드 구현
서브 페이지 : 상품 목록 생성 및 페이징, 상품 정렬 및 필터링 기능
디테일 페이지 : 상품 정보 제공 및 리뷰, QNA 등록 및 삭제 구현

상품 필터기능으로 원하는 상품의 데이터를 동적으로 받아오는 과정합니다. 원하는 상품을 찾을 때 까지 중복으로 필터를 적용시킬 수 있습니다.

배열 함수인 some을 사용하여 데이터에 있는 카테고리와 체크한 필터가 하나라도 일치한다면 상품이 출력되는 방식입니다.

그러기 위해서는 애니메이션적인 요소를 조화롭게 잘 사용해야 한다 생각합니다.

제이쿼리 활용 반응형 웹사이트 (빌보드 차트)

코딩 기간 : 2023년 3월 30일 ~ 4월 24일

HTML, CSS, JQUERY 사용

프로젝트 설명 및 후기

반응형 웹페이지 구현, 제이쿼리를 활용한 요소간의 동적인 상호작용 구현을 목표로 시작한 프로젝트입니다. 기존의 빌보드의 신문같은 디자인을 살리면서 음원 차트 순위를 직관적으로 사용자에게 전달하는 것을 목적으로 만들게 되었습니다.


미디어쿼리를 활용하여 어떤 디바이스든 예쁜 화면을 출력할 수 있도록 만들었습니다.

반응형 웹페이지를 만드는 것은 의외로 창의성을 요구하는 작업인 것을 느꼈습니다.

화면의 공간을 어떻게 써야할지 생각해야 하고, 사용자가 사이트의 의도를 잘 파악할 수 있을 지 계속 생각을 해야합니다.

코드보기

제이쿼리를 활용하여 요소들이 상호작용할 수 있도록 만들었습니다.

처음 사이트를 방문한 사용자라도 쉽게 기능을 사용할 수 있게 만드는 것이 가장 중요하다고 생각했습니다.

그리고 사이트의 미적요소를 해치지 않고 기능을 잘 구현하는 것 또한 신경을 써야겠구나 하고 생각한 프로젝트였습니다.
